# SBP fast pay
iOS приложение: https://link

Разработчик: Ilya Kuzmichev

### Ключевые слова
SBP, NSPK, C2B, C2C, СБП, НСПК, С2Б, система, быстрых, платежей, fast payments

### Назначение
Используйте приложение для тестирования функционала C2B-платежей в рамках СБП от НСПК

### Описание
Приложение позволяет считывать QR-коды, предназначенные для СБП-оплаты и отправлять их в тестовую систему оплаты. 
Вы можете использовать нашу тестовую среду или указать свой URL в remote settings (раздел custom). На указанный адрес по нажатии кнопки 'PAY' методом POST будет отправлено содержимое QR-кода в base64 кодировке в параметре 'qrData'.
Мы можем помочь Вам с интеграцией СБП C2B (а также C2C) для этого свяжитесь с нами через обратную связь.
