# SBP fast pay
iOS приложение: https://apps.apple.com/us/app/sbp-fast-pay/id1494174161?l=ru#?platform=iphone

Разработчик: Ilya Kuzmichev

### Ключевые слова
SBP, NSPK, C2B, C2C, СБП, НСПК, С2Б, система, быстрых, платежей, fast, payments, pay

### Назначение
Используйте приложение для проверки функционала C2B-платежей в рамках СБП от НСПК

### Описание
Приложение позволяет считывать QR-коды, предназначенные для СБП-оплаты, и отправлять их в систему проведения платежей, предназначенную для проверки интеграции. 
Вы можете использовать нашу систему проверки платежей или указать свой URL в remote settings (раздел custom). На указанный адрес по нажатии кнопки 'PAY' методом POST будет отправлено содержимое QR-кода в base64 кодировке в параметре 'qrData'.
Мы можем помочь Вам с интеграцией СБП C2B (а также C2C) для этого свяжитесь с нами через обратную связь.
